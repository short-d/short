# Short
URL shortening service

## Preview
![Demo](doc/demo.gif)

## Like `s/` extension?
Get it [here](https://github.com/byliuyang/short-ext)!

## Prerequisites
- Yarn v1.17.3
- Go   v1.12.7

## Getting Started

1. Build App
```bash
./bin/build
```

2. Start server
```bash
./build/short start --www build/web
```

3. Visit [http://localhost](http://localhost)
